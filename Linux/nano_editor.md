## usage memo

### exit/save
- save: Ctrl + o
- exit: Ctrl + x
- exit without saving: Ctrl + x + n
### copy/cut/paste
- copy: Alt + 6
- cut: Ctrl + k
FYI : current line will be selected if you define range selection.
